Timesheet Assist Bookmarklet

The "Timesheet Assist" bookmarklet has been created to make filling in the times and codes for the online timesheet a more simple and automated process. Add the bookmarklet to the browser bookmarks bar to get started.

Add the bookmarklet to the bookmarks bar

The bookmarks bar, "Favorites Bar" for IE, is a convenient place to store bookmarks that are used frequently. The bookmarks bar is located towards the top of the browser window, under the address bar. (See image below.)

If the bookmarks bar is not visible for your browser, click the link below that pertains to the browser that you are using and follow the steps to enable showing the bookmarks bar.

How to use the bookmarklet

In addition to functioning like a normal bookmark, navigating the browser to the online Timesheet page when the bookmarklet is clicked, the bookmarklet adds two key functions in assisting the user to populate their timecard. We'll examine both added functions next.

Auto-fill defaults:

If the bookmarklet was clicked to navigate the browser to the online timesheet page, click the bookmarklet again after the page has finished loading. A button with the text of, "Auto-fill defaults" is added to the web page next to the Oracle logo.

When the Auto-fill default button is clicked it will populate each weekday with 7.5 hours and code the time as Regular Work Hours. This action can be useful even when a few days vary from a standard work week. Click the Auto-fill defaults button to fill in the times with the standard 7.5 Regular Work Hours and then make updates to any days that vary from normal.

A dialog box will pop up when the Auto-fill default button is clicked and there are days that have already been populated with values.

To overwrite all of the previously populated days with the standard 7.5 Regular Work Hours, click the "OK" button. To cancel the Auto-fill defaults function, and leave all of the previously populated days unchanged, click the "Cancel" button.

Product Code:

The other functionality that the Timesheet Assist bookmarklet adds to help accurately update the timesheet is to automatically populate the five-digit "Product Code" whenever a "Time Reporting Code" is selected. When the "Time Reporting Code" is initially selected or changed, the corresponding Product Code is populated or changed to match the Time Reporting Code selection.
